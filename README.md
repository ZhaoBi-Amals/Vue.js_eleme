## 项目分析

#### 组件化开发
简单的讲，诸如页面中的‘⊕’标签、购物车、食物详情弹出框等等都是组件。哪里要用到它们，就在哪里引入它们，就像搭积木一样。我们要做的就是写好一个个的组件，再把这些组件一一加载在页面上。<br />
